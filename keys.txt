Filtering process
-----------------
From Table 2 in Galaxy DECaLS paper

Suggested cuts for elliptical: smooth > 0.7 and minimum number of votes = 4

Suggested cuts for spiral arms: spiral arms > 0.6 and minimum number of votes = 20

In pandas, if the file is called csv, then we can filter for elliptical galaxies as follows:

csv[(csv['smooth-or-featured_smooth_fraction'] > 0.75) &
    (csv['smooth-or-featured_smooth'] > 20) &
    (csv['merging_none_fraction'] > 0.75)
    ]

We include merging_none because we want the network to learn galaxies which are merging as well.

If we want to pick up the spirals, then

csv[(csv['has-spiral-arms_yes_fraction'] > 0.75) &
    (csv['has-spiral-arms_yes'] > 20)
    ]
